This Plugin makes the Thumbnails clickable with any URL you want.
There is an Extra URL Field within the Product.

== Description ==

This Plugin makes the Thumbnails clickable with any URL you want and open it in a new tab.

There will be new Field called "External URL" on the Product data. Just fill in any URL i.e. Affiliate URLs you want.
